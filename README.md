# Traffic Violation Detection System - Frontend

## Overview

This repository contains the React frontend for the AI-powered Traffic Violation Detection System.

The frontend allows users to upload road images, visualize detected traffic violations, and view analytics generated by the backend.

## Features

* Image Upload
* Detection Results
* Violation Analytics Dashboard
* Responsive UI

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS

## Installation

```bash
npm install
```

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```
